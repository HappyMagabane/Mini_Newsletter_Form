# Mini_Newsletter_Form
A responsive Newsletter subscription form built with HTML5 and external CSS, featuring input validation and topic selection.
📬 Mini Newsletter Subscription Form
A Clean & Functional Web Component
📝 Description
This project features a responsive and accessible Newsletter Subscription form. It was developed to practice structured HTML5 form handling and to demonstrate clean data entry design for web users interested in Web Development and Design.

🛠️ Tech Stack
HTML5: Used semantic form structures, including <label>, <input>, and <select> elements to ensure accessibility and proper data labeling.

CSS3: Integrated an external stylesheet (stylesheet.css) to maintain a professional "card-form" layout, practicing the industry-standard Separation of Concerns.

✨ Key Features
User Input Validation: Utilizes the required attribute to ensure users provide a name and a valid email address before submitting.

Dropdown Selection: Includes a category selector for topics like Web Development and Design, allowing for personalized user experiences.

Responsive Card Design: Styled with a specific .card-form class to ensure the component looks great as a standalone piece or integrated into a larger landing page.


💡 Learning Outcomes
Through this project, I strengthened my skills in:
Form Handling: Managing different input types (text, email, and select).
Project Organization: Linking external stylesheets properly to maintain clean HTML code.
UI/UX Basics: Designing a straightforward interface that guides the user toward a single clear action (Subscribing).
